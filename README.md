## ✨ Overview

Earthsome is a modern and scalable wellness marketplace built for brands focused on sustainable, plant-based, and cruelty-free products.  
The platform delivers a clean shopping experience with elegant UI, secure checkout, inventory management, and responsive performance.

---

## 🚀 Features

- 🌱 Wellness & Nutraceutical Products
- 🛍️ Modern eCommerce Experience
- 🔐 Secure Authentication System
- 📦 Inventory & Product Management
- 💳 Payment Gateway Integration
- 📱 Fully Responsive Design
- ⭐ Product Reviews & Ratings
- 📊 Admin Dashboard
- ⚡ Fast & Optimized Performance
